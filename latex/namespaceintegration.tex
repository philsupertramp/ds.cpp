Recursive implementation of numerical quadrature to approximate a given exact integral uses Simspon\textquotesingle{}s -\/ and Trapezoid Rule for approximation.

The Trapezoid Rule is given as

\textbackslash{}begin\{equation\} \textbackslash{}int \textbackslash{}limits\+\_\+a\texorpdfstring{$^\wedge$}{\string^}b f(x) dx \textbackslash{}approx T(x) = \textbackslash{}frac\{b -\/ a\}\{2\} \textbackslash{}left\mbox{[} f(a) + f(b) \textbackslash{}right\mbox{]} \textbackslash{}end\{equation\} Simpson is given as \textbackslash{}begin\{equation\} \textbackslash{}int \textbackslash{}limits\+\_\+a\texorpdfstring{$^\wedge$}{\string^}b f(x) dx \textbackslash{}approx S(x) = \textbackslash{}frac\{ b -\/ a \}\{ 6 \} \textbackslash{}left\mbox{[} f(a) + 4 f \textbackslash{}left( \textbackslash{}frac\{a+b\}\{2\} \textbackslash{}right)
\begin{DoxyItemize}
\item f(b) \textbackslash{}right\mbox{]} \textbackslash{}end\{equation\}
\end{DoxyItemize}

For approximation a tolerance is given, which is used to compare against the local error of approximation. The error is calculated as \$\$ \texorpdfstring{$\vert$}{|} S -\/ T \texorpdfstring{$\vert$}{|} \$\$ Whenever the error reaches the threshold of tol, the method has succeeded and the result of Simpson\textquotesingle{}s rule is returned.

{\bfseries{Note}} if (b-\/a) \texorpdfstring{$<$}{<} h\+Min we reached the minimal step with and haven\textquotesingle{}t found an satisfying result, therefore the algorithm exits with error code 1
